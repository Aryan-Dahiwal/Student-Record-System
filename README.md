# Student-Record-System
My first project in C.

A command-line application in C to manage student records.

## Features
- Add student records (name, roll number, marks)
- Display all students
- Search by roll number
- Delete a student
- Sort by marks

## How to Compile and Run
gcc main.c -o main
./main

## Tech Stack
- Language: C
- Concepts used: Structs, Linked Lists, File Handling
